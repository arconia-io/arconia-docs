# Arconia Documentation

Frameworks and tools for building cloud native software systems in Java with a superior developer experience.

## 📙&nbsp; Contributing

### Activating the development environment

This project uses [Flox](https://flox.dev) to manage development environments. To get started, you need to install Flox and activate the environment for this project. The main benefit of using Flox is that it allows you to run the project in a consistent environment without needing to install all dependencies on your local machine.

```shell
flox activate
```

When you're done with your work, you can deactivate the environment, and Flox will clean up all the packages and tools that were installed so that you won't have to worry about cluttering your system.

```shell
exit
```

### Building the website

```shell
task build
```
